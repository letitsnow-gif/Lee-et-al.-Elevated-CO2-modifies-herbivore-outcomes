# Lee-et-al.-Elevated-CO-modifies-herbivore-outcomes
R code and raw data associated with the paper (Lee et al., Elevated CO₂ modifies herbivore outcomes)
data/: Contains raw data files from the plant and herbivore bioassays, as well as the metabolomics analysis.
Ambient_or_Elevated_CO2_chamber/: Contains environmental logger data. Each subfolder corresponds to a specific growth chamber, and each file within represents a single day of logging data.
