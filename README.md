# Lee-et-al.-Elevated-CO-modifies-herbivore-outcomes
R code and raw data associated with the paper (Lee et al., Elevated CO₂ modifies herbivore outcomes)


data/: Contains raw data files from the plant and herbivore bioassays, as well as the metabolomics analysis.


Rscript: R code for the main analytical methods used in this study.


Ambient_or_Elevated_CO2_chamber: Contains environmental logger data. Each subfolder corresponds to a specific growth chamber, and each file within represents a single day of logging data.


Crops_Ambient_or_Elevated_CO2_chamber: Contains environmental logger data from experiments using Brassicaceae crops. Each subfolder corresponds to a specific growth chamber, and each file represents a single day of logging data. Please note that while data formats vary slightly due to the use of different logger models, all files consistently record CO₂, temperature, and relative humidity.
